# Cartes Figma - Instructions d'export

## Structure des dossiers

```
public/cards/
├── explore/          # Cartes de la section Explore
│   ├── map.png       # Carte interactive dynamique
│   ├── lightbulb.png # Découvertes autour de vous
│   └── globe.png     # Inspirations de voyages
├── share/            # Cartes de la section Share
│   ├── folder.png    # Album collaboratif
│   ├── chat.png      # Chat intégré par album
│   └── share.png     # Partage instantané
└── remember/         # Cartes de la section Remember
    ├── folder.png    # Souvenirs intelligemment regroupés
    ├── sparkles.png  # Récit interactif généré par l'IA
    └── chart.png     # Statistiques de voyage
```

## Instructions d'export depuis Figma

### Pour chaque carte :

1. **Sélectionner la carte** dans Figma
2. **Clic droit** → "Export" ou utiliser le panneau Export (à droite)
3. **Format recommandé** :
   - PNG : pour les cartes avec images/photos
   - SVG : pour les cartes vectorielles simples
   - Résolution : 2x ou 3x pour la qualité
4. **Nommer le fichier** selon le mapping ci-dessus
5. **Placer le fichier** dans le bon dossier (`explore/`, `share/`, ou `remember/`)

### Exemple pour la carte "Carte interactive dynamique" :

- Sélectionner la carte dans Figma
- Export → PNG @2x
- Nommer : `map.png`
- Placer dans : `public/cards/explore/map.png`

## Mapping des noms

### Section Explore
- `map.png` → Carte interactive dynamique
- `lightbulb.png` → Découvertes autour de vous
- `globe.png` → Inspirations de voyages

### Section Share
- `folder.png` → Album collaboratif
- `chat.png` → Chat intégré par album
- `share.png` → Partage instantané

### Section Remember
- `folder.png` → Souvenirs intelligemment regroupés
- `sparkles.png` → Récit interactif généré par l'IA
- `chart.png` → Statistiques de voyage

## Notes

- Les images seront automatiquement utilisées une fois placées dans les dossiers
- Format recommandé : PNG @2x (pour une bonne qualité sur tous les écrans)
- Taille approximative : 400x300px à 600x450px (ratio 4:3)
